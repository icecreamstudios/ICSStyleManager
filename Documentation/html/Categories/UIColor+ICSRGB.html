ics_colorWithRGBAValues:

Returns a color created with the given RGBA components. This is equivalent to calling ics_colorWithRGBValues:alpha: and passing rgbaValues.a as alpha parameter.

+ (UIColor *)ics_colorWithRGBAValues:(ICSRGBA)rgbaValues

Parameters

rgbaValues

The RGBA components of the new color.

Return Value

A color created with the given RGBA components.
